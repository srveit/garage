# garage
Control and monitor garage door opener using a Raspberry Pi Zero-W

Input Pins
|    name    |Phy|BCM|
|    light on| 21|  9|
|    motor up| 23| 11|
|  motor down| 31|  6|
|  open limit| 10| 15|
| close limit| 37| 26|
| door button| 35| 19|
|light button| 22| 25|
| lock switch| 29|  5|

Output pins
|      name      |Phy|BCM|
|     door button|  3|  2|
|     lock switch|  5|  3|
|    light button| 13| 27|
|     light relay| 16| 23|
| LED D3 (orange)|  7|  4|
|    LED D4 (red)| 11| 17|
